## Decent Network简介

Decent Network 构建一个去中心化的好友网络（基于 DHT）：身份即公钥，通信以“成为好友”为许可，在线状态与事件天然存在；并在其上叠加存储、数据访问与智能能力，让 Web3 + AI 应用无需重复搭建隐形基础设施即可快速上线与扩展。

过去很多 Web3 社交项目之所以难以形成规模，不是因为缺少协议或叙事，而是因为应用真正运行需要的关键基础能力长期缺失：即时互动、触达与关系边界、离线内容、数据访问、以及智能服务。区块链更像“凭据与状态的账本”，但社交与多数 AI 原生应用并不应该附属于某一条链，它们需要一套更完整、更稳健、可共享的底层平台。

Decent Network 提供应用必需的底座能力，并以网络化方式运行：  
- 即时通讯与实时互动  
- 存储（离线文件/媒体）  
- 账号/资料与权限边界  
- 数据访问（查询/索引/缓存/聚合）  
- 智能能力（模型访问、推理与执行能力的组合）

Beagle Chat 是 Decent Network 的旗舰应用形态之一（iOS/Android/Web），用于在真实用户场景中验证底层网络，并持续沉淀可复用的客户端框架（Beagle Chat Framework）。Decent Network 不限于社交：任何需要“互动 + 数据 + 智能”的应用都可以共享同一底座。
