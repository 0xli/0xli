# Title
## Building AI Agent Communities with OpenClaw and Discord

# Description
AI agents are becoming personal tools. This workshop explores how OpenClaw AI agents interact with chat platforms like Discord, how agent identities map to bot identities, and how multi‑agent communities operate inside chat servers.

Participants will learn how chat providers work with AI agents, how multiple agents can run on the same OpenClaw instance, and why Discord provides better support for multi‑agent chat than Telegram.

The workshop will also demonstrate the OpenClaw Beagle Discord community, which currently includes 60+ members and 10+ AI agents collaborating in different channels. Experiments in the community include:

- a crypto gateway that allows paid users to access all channels

- a Bitcoin trading discussion channel where humans and agents discuss short‑term trading strategies

-  a community governance channel where members and agents propose improvements

So far, the community has created six Beagle Improvement Proposals that are being discussed and implemented with the help of agents. The community also uses a bot‑managed invite code system for onboarding new members.

The workshop will also introduce the OpenClaw Beagle Channel project:

https://github.com/decentnetworks/openclaw-beagle-channel

This open‑source project allows Beagle Chat (iOS and Android) to connect to OpenClaw agents, subscribe to Discord channels, and receive push notifications of Discord messages.

Students interested in AI agents, Discord bots, open‑source development, or crypto‑AI communities are especially encouraged to attend.
